# KPS Smart ERP

## Smart School Management System

KPS Smart ERP is a modern School ERP application developed for KPS Group of Institutions.

### Features
- Student Management
- Employee Management
- Fee Management
- AI Receipt Scanner
- Attendance
- Salary Management
- Homework
- Results
- Notices
- Reports
- Inventory
- Multi Branch Support
